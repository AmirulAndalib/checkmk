`checkmk` ships several binaries and packages compiled for Windows, e.g. the Windows agent, its msi package or plugins like `mk-sql.exe`/`mk-oracle.exe`.
We are now switching the way we are signing those artifacts.

When looking at the certificate, users will notice the following difference.

* Before:
```
Subject: CN=Checkmk GmbH,O=Checkmk GmbH,ST=Bayern,C=DE
Issuer : CN=Sectigo Public Code Signing CA E36,O=Sectigo Limited,C=GB
```
* After:
```
Subject: CN=Checkmk GmbH,O=Checkmk GmbH,L=München,ST=Bavaria,C=DE
Issuer : CN=Microsoft ID Verified CS EOC CA 04,O=Microsoft Corporation,C=US
```
