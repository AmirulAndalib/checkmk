# Warn about Nagios Core being used in commercial editions

---------- | ---
date       | 2026-07-22T14:17:28.967806+00:00
version    | 3.0.0b1
class      | feature
edition    | pro
component  | checks
level      | 1
compatible | yes

Commercial editions of Checkmk are designed to run on the Checkmk Micro Core
(CMC). Running such a site on the Nagios core instead can lead to serious
performance problems, especially in larger environments, but so far there was
no signal pointing this out.

The <tt>OMD &lt;site&gt; performance</tt> service now reports <tt>WARN</tt> when
a commercial edition site is running the Nagios core, together with a message
recommending the CMC. Switching the site to the CMC clears the warning.

The community (raw) edition can only use the Nagios core and is therefore never
affected. Sites whose edition cannot be determined are left unchanged as well,
so no false positives are raised.
